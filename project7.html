Goal

The main goal of this project is to conduct a thorough evaluation of the games offered within the Xbox Game Pass Ultimate subscription. This assessment will include an analysis of their individual value propositions, categorization by genre, examination of ratings, and quantification of the total number of games. Ultimately, the project aims to provide valuable insights into the subscription's gaming content and its appeal to subscribers.

Main Questions

  1. How many games are included with Game Pass Ultimate?
  2. What is the total cost of all the games included?
  3. Of the games included, how many feature multiplayer options, and among them, how many offer local and online multiplayer modes? Are there games that offer both ?
  4. How are the games distributed based on their release years?
  5. How many games fall into each distinct age rating category, is the Game Pass Ultimate subscription appropriate for children?
  6. What features are included in Game Pass games, and how many games offer each of these features?
  7. How many games do you have in various genres, such as action & adventure, classics, indie and others?

Conclusions

  1. Game Pass Ultimate has a library of 466 games.
  2. The cumulative value of all the games featured in Game Pass Ultimate amounts to an impressive $12,988.85.
  3. Out of the 183 games supporting online multiplayer, 35 of them additionally feature local multiplayer options.
  4. Game Pass Ultimate offers a diverse collection of titles spanning from as early as 2001. The library is notably weighted towards more recent releases, with 90 games originating from 2022, while only 18 games date back to the period between 2001 and 2009.
  5. Game Pass Ultimate offers games across various age rating categories. It includes 173 games with a Mature (+17) rating, 90 games suitable for all ages, 80 games for those above 10, and 123 games designed for teenagers. This diversity makes Game Pass Ultimate an inclusive option for players of all age groups, ensuring there are age-appropriate games for children as well.
  6. Game Pass games come with a range of features. Notably, 145 games offer online co-op, 26 games provide local co-op, 64 games support Smart Delivery, 70 games offer 4K image quality, and 36 games have HDR support. These features enhance the gaming experience and cater to different player preferences.
  7. Game Pass Ultimate presents a diverse selection of games spanning various genres. Notably, the "action & adventure" genre boasts the largest offering with 360 games, making it a dominant category within the subscription. Additionally, the "indie" genre features 126 titles, and the "role-playing" genre comprises 122 games, establishing Game Pass as an excellent choice for those seeking to explore the indie gaming scene. The "classics" genre, while more limited, still offers 14 titles for those interested in revisiting or discovering older game classics.

Data extracting:
    Python
  • Developed a Python script powered by Selenium to gather valuable insights from the Xbox website. This information pertains to Game Pass Ultimate, incorporating game titles, release years, and estimated values. To explore the web scraper and its accompanying documentation click here
Data cleaning:
    Tableau
  • The data was initially imported into Tableau, with careful attention to correctly specifying the data types for fields such as Game ID, features, genres, and platforms.
  • In order to enable effective analysis, relationships between the tables were established, with the Game ID serving as the primary key. This ensured that data from various sources could be interconnected seamlessly.
  • Furthermore, data was pivoted to reorganize features, genres, and platforms, enhancing the suitability of the data for the creation of charts and visualizations.
