Goal

A different perspective on Portuguese wine (is Portuguese wine known, have high quality and cheap price?, what provinces are most well-known?, what varieties the wine tasters are most fond of?), as well new suggestions of high quality wine that is not expensive.

Main Questions

  1. Is Portuguese wine popular among the wine tasters?
  2. Do more expensive wines have higher number of points?
  3. Does Portuguese wine have high points compared with the rest of the world?
  4. Is Portuguese wine expensive comparing with the rest of the world?
  5. Which provinces are the most popular?
  6. What are the best Portuguese variants?
  7. What are the best wines with the cheapest price possible?

Reports

Conclusions

  1. Considering WineEnthusiast is a company based in the USA, Portugal composing 4.57% of the total tweets and being the 5th country with the most tweets can be consider good and a proper sample study.
  2. A growth in points (with some exceptions) can be observed with the raise of the price until it reaches 127$; from there points oscillate between the maximum value for prices such as 450$ and 650$ and values bellow the Portuguese average for 790$.
    More expensive wine usually has a better quality but only up to a certain point. The price may also affect what the taster expects which may lead to higher prices having lower classifications.
    In conclusion price may affect the quality of the wine but it is not the only factor.
  3. Portuguese average points is slightly higher than the global average.
  4. On average a Portuguese wine is cheaper than other wines, nonetheless, the cheapest Portuguese wine is not the cheapest wine globally and neither is the most expensive one being 2300 cheaper than the most expensive wine globally.
  5. Douro, Alentejo, Tejo, Lisboa, Vinhos Verdes are the most popular Portuguese provinces.
  6. The varieties with the highest amount of points are used in white, red and port wines mostly. They are Terrantez, Sercial, Ramisco, Bual and Bordeaux-style red blend.
  7. The best wines when it comes to high points and cheap price are:
    • Adega Cooperativa do Cartaxo 2012 Xairel Red (Tejo)
    • Herdade dos Machados 2011 Toutalga Tinto Red (Alentejano)
    • Campelo 2009 Tinto Red (Vinho Verde)
    • Adega Cooperativa da Batalha NV D'Ó Red (Portuguese Table Wine)
    • Campelo 2009 Rosé (Vinho Verde)
    • Herdade dos Machados 2009 Morgado da Canita Red (Alentejano)
    • Adega Cooperativa Ponte de Barca 2010 Escolha Rosé (Vinho Verde)
    • Lagosta 2008 White (Vinho Verde)
    • Caves Velhas 2015 Casaleiro Colheita Selecionada Red (Tejo)
    It’s important to highlight the fact that the tweets were taken from a company based in the United States and these conclusions reflect their vision on Portuguese wine and are not about Portuguese wine in general.

Database Description

Dataset containing ~130k wine reviews published to WineEnthusiast, including taster name, wine price, variety, score, country, county, vineyard, etc.

  • Database Source: Maven Analytics
  • Data Structure: Single table
  • # of records: 129,971
  • # of fields: 14
  • Date: 09/10/2020
  • File type: CSV

More details

Data cleaning:
  • MS Excel

    • There were a lot of different values for the same province. In order to extract accurate results, the starting point is to get the actual values for all Portuguese provinces; after that select for each of the values on the data base which would be the most fitting province, created a color relation between both and then added them together by color in order to have the actual amounts of wine per province. I use this specific method because this database had only this purpose to me and I would not use it again.

    Normally this process of data cleaning would have been done in the whole data base but since only Portuguese provinces were important to take my conclusions I decided to do it this way.

  • MSSQL

    • Eliminate unpaired values of price and points, that would cause problems on taking conclusions on the dataset, by selecting only rows where both values (price and points) were not null.

SQL queries: 

        -- Select total no of tweets --
                SELECT 
                    COUNT(id) AS total_tweets
                FROM dbo.winetasting
                WHERE id IS NOT NULL;
                        
        -- Select no_of_tweets per country --
                        
            SELECT 
                country,
                COUNT(id) AS no_of_tweets
            FROM dbo.winetasting
            WHERE id IS NOT NULL
            GROUP BY country
            ORDER BY 
                no_of_tweets DESC;

        -- select price and points from portuguese wines--
            SELECT 
                points,
                price
            FROM dbo.winetasting
            WHERE 
                price IS NOT NULL 
                AND points IS NOT NULL
                AND country = 'Portugal';

        -- Select global average points --
            SELECT 
                AVG(points) AS avg_points_global
            FROM dbo.winetasting;
                    
        -- Select portuguese average points --
            SELECT
                AVG(points) AS avg_points_portugal
            FROM dbo.winetasting
            WHERE country = 'Portugal';
        
        -- Select Average, maximum and minimum price globally --
            SELECT 
                MAX(price) as max_price,
                AVG(price) as avg_price,
                MIN(price) as min_price
            FROM dbo.winetasting;
            
        -- Select Average, maximum and minimum price for Portugal --
            SELECT 
                MAX(price) as max_price,
                AVG(price) as avg_price,
                MIN(price) as min_price
            FROM dbo.winetasting
            WHERE country='Portugal';
        
        --Select provinces when the ones with the bigger no of wines come of top--
            SELECT
                province,
                COUNT(DISTINCT(title)) AS no_of_wines
            FROM dbo.winetasting
            WHERE country = 'Portugal'
            GROUP BY 
                province
            ORDER BY 
                no_of_wines DESC;
        
        /* Select varieties with the ones with 
        the highest average of points coming of top */ 
                
            SELECT
                variety,
                AVG(points) AS avg_points
            FROM dbo.winetasting
            WHERE country = 'Portugal'
            GROUP BY 
                variety
            ORDER BY 
                avg_points DESC; 

        /* Select portuguese wines with the
        higher amount of points and lower price*/
                
            SELECT 
                title,
                points / price AS ratio
            FROM dbo.winetasting
            WHERE 
                country = 'Portugal'
                AND price IS NOT NULL
                AND points IS NOT NULL
            ORDER BY 
                    ratio DESC;
