Goal

Netflix is an American subscription video on-demand streaming service and production company. The main goal is to find out how Netflix’s content changed over time (2016-2021) and what content Netflix has available based on rating and category.

Main Questions

  1. Have the proportion between TV shows and movies changed?
  2. Is Netflix new content actually new?
  3. What are the most popular categories?
  4. Does Netflix include contents for every age-group ?

Conclusions

  1. Netflix has consistently expanded its content offerings over time. A recurring characteristic of its library is the imbalance between movies and TV shows, as there are always more movies than TV shows available on the platform.
  2. The majority of TV shows on Netflix were not added to the platform in the year they were originally released. The majority of older content on the platform, however, are movies.
    The addition of new content to Netflix has seen steady growth until 2020, but there was a noticeable drop in releases in 2021.
  3. Netflix offers a diverse range of content, divided into approximately 40 categories. Drama, Comedy, and Action-Adventure are the most prevalent categories across all content on the platform. However, if we delve deeper, we find that Documentaries are the third most popular movie category, while Crime is the third most popular TV show category.
  4. Netflix provides content for audiences of all ages. However, only 3% of the content is labeled as suitable for a general audience, indicating that it may not be the ideal platform for those seeking content specifically for children.

Data cleaning:
  • MS Excel

    • Start by removing columns with information that is not relevant for the goal (cast, director).
    Column: show_id
    Remove 's' from the value in order to obtain an integer
    Column: type_id
    Using the function IFS substitute the type for it's corresponding type_id
    Column: date_added
    Change the date format to YYYY-MM-DD in order to be compatible with SQL
    Column: rating_id
    Select distinct values. Turn distinct values with the same meaning into one value ex: TV-G/G. Order ratings by age ascending. Create a number key for each distinct_value(rating_id). Replace each rating with it's corresponding id.
    Column: duration
    Sort a copy of the database by type_id. Divide between movies and TV shows. Remove min, seasons, season from the values. Clean the duration column. Create a column named no_of_seasons. Copy the values of duration and no of seasons to it's corresponding columns. Sort by show_id.
    Column: categories
    Split the listed_in column into the different categories(columns:category1_id, category2_id and category3_id). Replace the categories by they corresponding value in the categories table.

SQL queries: 

        -- select no of shows by type and by year--

            SELECT 
                YEAR(nt.date_added),
                t.type_name,
                COUNT(nt.show_id) 
            FROM dbo.netflix_titles AS nt
                LEFT JOIN dbo.types AS t
                    ON (nt.type_id=t.type_id)
            GROUP BY 
                YEAR(nt.date_added),
                t.type_name
            ORDER BY 
                 YEAR(nt.date_added) ASC;
                        
                        
        -- select from all content the amount that was added on its release year --
                        
            SELECT
                YEAR(nt.date_added),
                t.type_name,
                COUNT(CASE
                        WHEN nt.release_year=YEAR(nt.date_added) 
                            THEN 1 
                        END) AS new_content,
                COUNT(nt.show_id) AS total_content		
            FROM dbo.netflix_titles AS nt
                LEFT JOIN dbo.types AS t
                    ON (nt.type_id=t.type_id)
            GROUP BY 
                YEAR(nt.date_added),
                t.type_name
            ORDER BY 
                YEAR(nt.date_added) ASC;
                        
                        
        -- select no of movies per category--
                        
                        
            DECLARE @Category INT 
            SET @Category=1
            WHILE ( @Category <= 48)
            BEGIN
                SELECT
                    @Category AS category_id,
                    SUM(CASE 
                        WHEN category1_id = @Category OR category2_id = @Category 
                            OR category3_id = @Category  THEN 1 
                        ELSE 0 
                        END) AS no_of_movies
                FROM dbo.netflix_titles
                WHERE type_id=1
                SET @Category = @Category + 1
             END;
                        
                        
        -- select no of tv shows per category--
                        
                        
            DECLARE @Category INT 
            SET @Category=1
            WHILE ( @Category <= 48)
            BEGIN
                SELECT
                    @Category AS category_id,
                    SUM(CASE 
                        WHEN category1_id = @Category OR category2_id = @Category 
                            OR category3_id = @Category  THEN 1 
                        ELSE 0 
                        END) AS no_of_tvshows
                FROM dbo.netflix_titles
                WHERE type_id=2
                SET @Category = @Category + 1
            END;
                            
                        
        -- select shows by type,rating and year --
                        
            SELECT
                YEAR(nt.date_added),
                r.rating,
                COUNT(CASE 
                        WHEN nt.type_id=1 THEN 1 END) as no_of_movies,
                COUNT(CASE 
                        WHEN nt.type_id=2 THEN 1 END) as no_of_tvshows
            FROM dbo.netflix_titles AS nt
                LEFT JOIN dbo.ratings AS r
                    ON (nt.rating_id=r.ratings_id)
            GROUP BY 
                YEAR(nt.date_added),
                 r.rating
            ORDER BY 
                YEAR(nt.date_added) ASC;
